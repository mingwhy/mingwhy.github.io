部署到GitHub

参考 搭建个人博客网站教程-07：Hexo+GitHub Pages：将网站部署到GitHub Pages上
在Github中创建目录仓库
个人经验：在创建new repo时候，一定要勾选“Add a README file”，不然后面会看不到GitHub Pages的域名和部署分支
创建之后，通过“Settings”，进入GitHub Pages相关的部分，确定branch是main。
此时在浏览器中输入，https://GitHub用户名.github.io 应该会显示最简单的博客页面。

安装 Hexo 的扩展 hexo-deployer-git

在博客文件夹下执行：npm install hexo-deployer-git --save
编辑博客安装文件夹下的_config.yml文件，翻到底部，修改deploy部分为：

在_config.yml文件中修改为repo: https://[token]@github.com/GitHub用户名/GitHub用户名.github.io.git.
