在Github中创建目录仓库
个人经验：在创建new repo时候，一定要勾选“Add a README file”，不然后面会看不到GitHub Pages的域名和部署分支
创建之后，通过“Settings”，进入GitHub Pages相关的部分，确定branch是main。
此时在浏览器中输入，https://GitHub用户名.github.io 应该会显示最简单的博客页面。
